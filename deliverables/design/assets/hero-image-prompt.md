# Hero Image Prompt

## Option 1: Product Screenshot (Recommended)

**不需要 AI 生成图。** 直接使用真实产品截图：

1. 打开 tonetool.org
2. 设置频率为 440Hz，波形为 Sine
3. 点击 Play，让波形可视化显示
4. 截图工具面板区域（包含 Canvas 波形、频率显示、控件）
5. 在 Figma/Photoshop 中包装：
   - 添加 subtle shadow (0 20px 60px rgba(0,0,0,0.5))
   - 添加 1px border (#1E1E2E)
   - 圆角 20px
   - 背景透明或 `#08080F`

这是最佳方案 — 真实 > 虚假。

# tonetool.org 冷启动渠道计划

## 推荐渠道（首周选 2-3 个）

### 🥇 第一梯队（高流量 + 精准用户）

#### 1. Reddit（推荐首选）
- **r/audiology** — 听力专业人士
- **r/headphones** — 耳机爱好者
- **r/audio** — 音频通用
- **r/WeAreTheMusicMakers** — 音乐制作人
- **r/LifeProTips** — 如果有独特卖点

**发帖格式：**
```
Title: I built a free online tone generator with 11 audio testing tools — no signup, works in browser

Body: [简短介绍 + 工具列表 + 链接]
```

**注意：** Reddit 反感硬广，要以「分享项目」的角度发帖

#### 2. AI/工具目录站
- **There's An AI For That** (theresanaiforthat.com)
- **AI Tool Directory** (aitoolsdirectory.com)
- **Futurepedia** (futurepedia.io)
- **ToolPilot** (toolpilot.ai)

**提交方式：** 大多数有 Submit 按钮，填写表单即可

#### 3. Product Hunt
- 需要提前准备
- 最佳发布时间：周二/周三 上午 12:01 PT
- 需要有人「maker」发帖

### 🥈 第二梯队（SEO + 长尾流量）

#### 4. Hacker News（选 Show HN）
- 适合技术向用户
- 标题格式：`Show HN: Free online tone generator with 11 audio tools`

#### 5. Indie Hackers
- 适合独立开发者社区
- 可以写「我是如何做的」故事

#### 6. 周刊投稿
- **AI Tool Weekly**
- **Ben's Bites**
- **The Rundown AI**

### 🥉 第三梯队（补充）

#### 7. X/Twitter
- 发 thread 介绍项目
- @相关音频博主

#### 8. YouTube
- 录一个 60 秒演示视频
- 标题包含关键词

## 执行节奏

| 时间 | 动作 |
|------|------|
| Day 0 | 提交目录站 + Reddit 发帖 |
| Day 1-2 | 回复评论 + 调整 |
| Day 3-5 | Product Hunt + HN |
| Day 5-7 | 周刊投稿 + X/Twitter |
